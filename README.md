# Rekam Medis - Klinik Ciruas

Sistem Informasi Rekam Medis untuk Klinik Ciruas.
Dibangun menggunakan Laravel dan Inertia.js (React).

## Fitur
- Pendaftaran Pasien
- Rekam Medis
- Poliklinik
- Apotek / Obat
- Laporan & Statistik
